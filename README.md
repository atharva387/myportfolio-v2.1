## 🔧 Installation & Setup

To run this project locally, follow these steps:

```bash
# Clone the repository
git clone https://github.com/atharva387/myportfolio-v2.1.git
cd myportfolio-v2.1

# Install dependencies
npm install

# Start the development server
npm run dev
